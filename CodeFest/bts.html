Week 1 | Getting Started

Pre-Week1: Before the official start of Codefest, we formed our group and communicated on WhatsApp where we decided our project concept: an educational game + website about the pandemic.
In the first week, we then started working out details, game concepts, art style, and collected our ideas on Google Doc. Later, we switched over to Discord so that we could more easilly share images and group call. A website framework which the game is intended to be hosted on is created pushed onto Github.

Preparation:

  • Set up a UnityID, UnityHub, and Unity Version 2019.4.17f1
  • Look for and/or create assets
  • Look up and follow online tutorials

Our Intial Ideas:

  • Game about epidemics and contact tracing
  • Target Audience: General Public
  • Point and Click Style game
  • First Person Perspective
  • Need concrete objectives
  • End Goal: Limit the spread of the virus using Covid-19 data
  • The player clicks on textboxes to make a decision
  • When the player makes a choice, there will be a popup that will explain why the selected choice was good or bad
  • The player's choices affect future events
  • The player can play as a person in power

Week 2 | Visuals and Gameplay

We now focus on the gameplay. Roles are individually assigned based on our skillset, experience, and interest. The concept of the game is a player surving the pandemic in their home. We then flesh out the player's roles, screen, and the rooms. Creation of the game on Unity starts and the website is filled with material to organize and supplement the game's educational information.

Ideas:

  • Point and Click
  • Player will play as a journalist
  • Player Perspective: Side of the Room | First Person | 2D
  • Pixel Art Style
  • Mouse Input only
  • Include an inventory system
  • Aspect Ratio: 16:9 | Background Image: 300x200 pixels
  • Include infecton chance percentage
  • Could possibly shift to a Visual Novel Style game
